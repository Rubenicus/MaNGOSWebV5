1. First of all, you must create an account. The account is used to log into both the game and our website. Click here to open the registration page.

2. Install World of Warcraft. You can download the Windows client from HERE or the Mac client from HERE.

3. Extract the contents where you would like them to go.

4. Go into the World of Warcraft folder you just extracted and open up the file called "realmlist.wtf" with a text editor such as Notepad. To do this, you must right click on the file and choose properties, then select notepad as the default software for files with the ".wtf" ending. You may also just start the text editor and drag the file into the edit window.

5. Erase all text and change it to:

set realmlist forgottenwow.com

You may now start playing! If you need any help, join our Discord and someone will be able to assist you ASAP.
